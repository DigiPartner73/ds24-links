3. TikTok API — Datenzugriff und Berechtigungen

Die App nutzt folgende TikTok API-Berechtigungen:

  • user.info.basic — Lesen grundlegender Profilinformationen (User-ID, Anzeigename, Avatar)
  • video.publish — Veröffentlichung von Videoinhalten in Ihrem Namen
  • video.upload — Hochladen von Videoinhalten auf TikTok

Wir rufen ausschließlich die Daten ab, die zur Erfüllung der App-Funktionen notwendig sind. Wir verkaufen oder teilen Ihre TikTok-Daten nicht mit Dritten.

4. Speicherung und Verarbeitung

Die erhobenen Daten werden wie folgt gespeichert:

  • Airtable (USA) — Speicherung von Produktdaten und Veröffentlichungsprotokollen. Airtable ist nach dem EU-US Data Privacy Framework zertifiziert.
  • Dropbox (USA) — Speicherung von Audio- und Videodateien. Dropbox ist nach dem EU-US Data Privacy Framework zertifiziert.
  • Make.com (EU) — Automatisierungsplattform zur Verarbeitung der API-Aufrufe.

TikTok-Zugriffstoken werden verschlüsselt gespeichert und nicht an Dritte weitergegeben. Sie werden ausschließlich zur Authentifizierung gegenüber der TikTok API verwendet.

5. Datenweitergabe an Dritte

Wir geben Ihre Daten nicht an Dritte weiter, außer:

  • An TikTok, soweit dies zur API-Nutzung erforderlich ist

7. Widerruf der TikTok-Autorisierung

Sie können die Autorisierung der App jederzeit widerrufen:

  • TikTok App öffnen → Profil → Einstellungen → Sicherheit → Autorisierte Apps → DS24 Affiliate Automation → Zugriff entziehen

Nach dem Widerruf werden keine weiteren Daten über die TikTok API abgerufen. Bereits gespeicherte Daten werden auf Anfrage gelöscht.
