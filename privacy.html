2. Zweck der Anwendung

DS24 Affiliate Automation ist ein automatisiertes Tool zur Erstellung und Veröffentlichung von Affiliate-Marketing-Videoinhalten auf sozialen Plattformen, darunter TikTok, Pinterest, Instagram und YouTube. Die App nutzt die offiziellen APIs dieser Plattformen, um Inhalte im Namen autorisierter Nutzer zu verwalten und zu veröffentlichen.

4. Pinterest API — Datenzugriff und Berechtigungen

Die App nutzt folgende Pinterest API-Berechtigungen:

  • boards:read — Lesen von Board-Informationen des autorisierten Kontos
  • boards:write — Erstellen und Verwalten von Boards
  • pins:read — Lesen von Pin-Informationen
  • pins:write — Erstellen und Veröffentlichen von Pins im Namen des Nutzers
  • user_accounts:read — Lesen grundlegender Kontoinformationen

Pinterest-Zugriffstoken werden verschlüsselt gespeichert und ausschließlich zur Authentifizierung gegenüber der Pinterest API verwendet. Sie werden nicht an Dritte weitergegeben.

5. TikTok API — Datenzugriff und Berechtigungen

Die App nutzt folgende TikTok API-Berechtigungen:

  • user.info.basic — Lesen grundlegender Profilinformationen (User-ID, Anzeigename, Avatar)
  • video.publish — Veröffentlichung von Videoinhalten in Ihrem Namen
  • video.upload — Hochladen von Videoinhalten auf TikTok

6. Speicherung und Verarbeitung

Die erhobenen Daten werden wie folgt gespeichert und verarbeitet:

  • Airtable (USA) — Speicherung von Produktdaten, Veröffentlichungsprotokollen und API-Token. Airtable ist nach dem EU-US Data Privacy Framework zertifiziert.
  • Dropbox (USA) — Speicherung von Audio- und Videodateien. Dropbox ist nach dem EU-US Data Privacy Framework zertifiziert.
  • Make.com (EU) — Automatisierungsplattform zur Verarbeitung der API-Aufrufe.

Alle Zugriffstoken werden verschlüsselt gespeichert und nicht an Dritte weitergegeben. Sie werden ausschließlich zur Authentifizierung gegenüber den jeweiligen Plattform-APIs verwendet.

9. Widerruf der Autorisierung

Sie können die Autorisierung der App jederzeit bei der jeweiligen Plattform widerrufen:

  • Pinterest: Pinterest App → Einstellungen → Sicherheit und Datenschutz → Apps und Websites → DS24 Affiliate Automation → Zugriff entfernen
  • TikTok: TikTok App → Profil → Einstellungen → Sicherheit → Autorisierte Apps → DS24 Affiliate Automation → Zugriff entziehen
  • Instagram/Meta: Facebook → Einstellungen → Apps und Websites → DS24 Affiliate Automation → Entfernen
  • YouTube/Google: myaccount.google.com → Sicherheit → Drittanbieter-Apps mit Kontozugriff → DS24 Affiliate Automation → Zugriff entfernen

Nach dem Widerruf werden keine weiteren Daten über die jeweilige API abgerufen. Bereits gespeicherte Daten werden auf Anfrage gelöscht.
