2. Zweck dieser Anwendung

Diese Anwendung nutzt die Pinterest API v5, um automatisiert Affiliate-Marketing-Inhalte (Pins) auf Pinterest zu veröffentlichen. Die veröffentlichten Inhalte bewerben Produkte von Affiliate-Netzwerken (insbesondere Digistore24 und ClickBank). Alle veröffentlichten Posts sind als Werbung gekennzeichnet.

3. Erhobene Daten und Verwendung

3.1 Pinterest-Daten (über die Pinterest API)

Im Rahmen der Nutzung der Pinterest API werden folgende Daten verarbeitet:

  • Pinterest-Konto-ID und öffentliche Profilinformationen des verbundenen Business-Accounts
  • Board-IDs und Board-Namen des eigenen Pinterest-Kontos
  • Pin-Metadaten: Titel, Beschreibung, Bild-URL, Ziel-Link (Affiliate-Link)
  • Pin-Performance-Daten: Impressionen, Klicks, Saves (nur für eigene Pins)

Alle über die Pinterest API abgerufenen Daten beziehen sich ausschließlich auf den eigenen Pinterest Business-Account und werden nicht an Dritte weitergegeben.

3.2 Intern gespeicherte Daten

Die Anwendung speichert intern in einer Airtable-Datenbank:

  • Produkt-Metadaten (Produktname, Kategorie, Affiliate-Link)
  • Generierte Texte und Bilder für Social-Media-Posts
  • Veröffentlichungsstatus und Zeitstempel
  • Performance-Metriken der eigenen Posts (Klicks, Impressionen)

Es werden keine personenbezogenen Daten von Dritten (Endnutzern) gespeichert.

3.3 Click-Tracking

Affiliate-Links werden über eine Redirect-Seite (go.html) geleitet, um Klicks zu zählen. Dabei wird nur der Zeitstempel und die Produkt-ID gespeichert — keine IP-Adressen, keine Nutzerkennungen, keine personenbezogenen Daten.

4. Weitergabe von Daten

Daten werden nicht an Dritte verkauft oder zu Werbezwecken weitergegeben. Eine Weitergabe erfolgt ausschließlich an:

  • Pinterest: Im Rahmen der API-Nutzung gemäß den Pinterest Datenschutzrichtlinien
  • Airtable: Als Datenbank-Provider
  • Anthropic (Claude API): Für die KI-gestützte Texterstellung (keine personenbezogenen Daten)

5. Datenspeicherung und -löschung

Daten werden so lange gespeichert, wie sie für den Betrieb des Systems erforderlich sind. Eine Löschung kann jederzeit auf Anfrage erfolgen.
